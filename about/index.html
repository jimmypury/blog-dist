主要技术栈

本站是一个由 Hexo 生成的博客。

  • 使用 Hexo 8 构建站点，CI 环境使用 Node.js 20。
  • Markdown 由 hexo-renderer-marked 渲染，模板使用 EJS，样式使用 Stylus。
  • 代码高亮使用 highlight.js，数学公式使用 MathJax。
  • 评论系统使用 giscus，数据保存在独立的 GitHub Discussions 仓库中。

主题与前端开源项目

本站主题基于 Pieter Robberechts 开发、采用 MIT 许可证的 Hexo Theme Cactus，并针对各种功能进行了修改。

主题和页面还使用了以下开源项目：

  • jQuery：基础 DOM 操作及部分主题交互。
  • clipboard.js：代码块复制功能。
  • Font Awesome：站点图标。
